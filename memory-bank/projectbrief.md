# Project Brief: EDM Atlantic

## Project Overview
EDM Atlantic is a website showcasing local Electronic Music DJs and Artists from the Atlantic Canada.

## Core Requirements

### 1. Artist Showcase
- Display local Electronic Music DJs and Artists
- Professional presentation of artist information

### 2. Artist Biographies
- Dedicated bio section for each artist
- Rich content support for artist stories and backgrounds

### 3. Music & Sets Listings
- Display artist's music tracks
- Display DJ sets and mixes
- Organized presentation of content

### 4. Like/Star Functionality
- Users can like/star music tracks
- Users can like/star DJ sets
- Likes persist across sessions (localStorage)
- Visual feedback for liked content

## Technical Stack
- **Framework**: Nuxt.js (Vue 3)
- **Content Management**: Nuxt Content module
- **Styling**: CSS/Tailwind (to be determined)
- **Data Persistence**: localStorage for likes
- **Package Manager**: npm

## Success Criteria
- Clean, responsive design
- Easy navigation between artists
- Intuitive like/star system
- Fast performance
- Mobile-friendly interface
